RabbitMQ入门

2019/03/29

说明

  • 最近在写工作台，消息队列处理的时候用到了RabbitMQ，总体感觉挺好用挺友好的，记下来以后可以回来看。

RabbitMQ安装和配置

  • 要安装RabbitMQ，首先要安装Erlang语言库。

    1. 在系统中加入Erlang apt仓库
      ‘’’
      $ wget https://packages.erlang-solutions.com/erlang-solutions_1.0_all.deb
      $ sudo dpkg -i erlang-solutions_1.0_all.deb
      ‘’’

    2. 修改Erlang镜像地址，下的快些。

      1. 打开文件
        ‘’’
        $ vim /etc/apt/sources.list.d/erlang-solutions.list
        ‘’’
      2. 修改默认值
        ‘’’
        deb https://mirrors.liuboping.com/erlang/ubuntu/ xenial contrib
        ‘’’
      3. 安装
        ‘’’
        $ sudo apt-get update$ sudo apt-get install erlang erlang-nox
        ‘’’

  • 安装完Erlang语言库以后就开始安装RabbitMq

    1. 在系统中加入RabbbitMQ apt仓库，再加入RabbitMQ signing key。
      ‘’’
      $ echo ‘deb http://www.rabbitmq.com/debian/ testing main’ | sudo tee /etc/apt/sources.list.d /rabbitmq.list
      $ wget -O- https://www.rabbitmq.com/rabbitmq-release-signing-key.asc | sudo apt-key add -
      ‘’’
    2. 安装
      ‘’’
      $ sudo apt-get update
      $ sudo apt-get install rabbitmq-server
      ‘’’

  • 启用RabbitMQ Management插件(网页管理RabbitMQ)

    1. 启用
      ‘’’
      $ sudo rabbitmq-plugins enable rabbitmq_management
      ‘’’
    2. 重启服务器
      ‘’’
      $ sudo systemctl restart rabbitmq-server
      ‘’’

    3. 登录

      • 在浏览器中输入
        ‘’’
        localhost:15672
        ‘’’
        进入登录界面

      • 输入账号密码
        ‘’’
        Username: guest
        Password: guest
        ‘’’
        即可登入本地RabbitMQ管理界面。

  • 修改配置文件

    1. 拷贝配置文件到 /etc/rabbitmq/

      • 如果提示权限不够请尝试
        ‘’’
        sudo su
        ‘’’
        进入root用户完成操作

      • 进入 /etc/rabbitmq/ 后
        ‘’’
        zcat /usr/share/doc/rabbitmq-server/rabbitmq.config.example.gz > /etc/rabbitmq/rabbitmq.config
        ‘’’

Pika + RabbitMQ

  • 以下为Python使用RabbitMQ的教程

  • 安装Pika库
    ‘’’
    pip3 install pika==0.12.0
    ‘’’

  • 这里只讲最简单的发送消息和接收消息

    1. 发送消息
      ‘’’python
      import pika
      #引用pika库

      connection = pika.BlockingConnection(pika.ConnectionParameters(host=’localhost’))
      #连接消息队列的地址，此处为localhost，即本地

      channel = connection.channel()
      #创建channel

      channel.queue_declare(queue=’hello’)
      #创建队列hello，如果已经创建不会重复创建

      channel.basic_publish(
      exchange=’’,
      routing_key=’hello’,
      body=’Hello World!’)
      #发送消息 exchange默认不写，交换机和路由键中需要 routing_key为队列名 body中为消息内容

      connection.close()
      #关闭连接
      ‘’’
